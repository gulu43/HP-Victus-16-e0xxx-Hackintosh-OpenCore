# HP Victus 16-e0xxx Hackintosh OpenCore

## Specs
- CPU: AMD Ryzen 5 5600H
- GPU: RX 5500M (disabled) + Radeon iGPU
- RAM: 16GB DDR4 3200MHz
- Storage: WD Blue SN5000 500GB
- WiFi: (original unsupported, replace with AX200NGW)
- macOS: Tahoe 26
- OpenCore: 1.0.x

## What Works
✅ Display
✅ Audio (VoodooHDA)
✅ Trackpad
✅ Battery
✅ USB ports
✅ Ethernet
✅ Brightness keys

## What Doesn't Work
❌ WiFi (replace card with AX200NGW)
❌ Bluetooth (replace card)
❌ SD Card reader
❌ Microphone
